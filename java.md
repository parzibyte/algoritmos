# Compilación
Si quieres puedes usar NetBeans y evitar compilar manualmente. También puedes hacerlo por consola. Suponiendo que tienes un
archivo llamado **Main.java** podrías compilarlo con:

`javac Main.java`

Y después ejecutarlo con:

`java Main`
